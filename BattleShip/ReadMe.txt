How to generate binary files:
	gcc -Wall comserv.c gestpart.c -o game
	#The above comand will make the binary of administrator
	
	gcc -Wall comclient.c ihm.c -o player
	#The above comand will make the binary of Player  
	
	
	to Execute write 
		./Player (As allowed number of players to make in different terminals).
		
		./game (To make an administrator terminal)
		
		
		
		
Note: First execute administrator then player.

Players are registered in users.txt <name>@<password>  (@ is the separator)
